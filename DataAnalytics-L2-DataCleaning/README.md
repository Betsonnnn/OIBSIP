# Medical Appointment Data Cleaning

## Oasis Infobyte Data Analytics Internship

### Task 2: Data Cleaning

## Project Overview

This project focuses on cleaning the Medical Appointment dataset by identifying and correcting data quality issues. The dataset was examined for missing values, duplicate records, invalid entries, and incorrect data types to prepare it for analysis.

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- VS Code
- Jupyter Notebook

## Data Cleaning Performed

- Checked for missing values
- Checked duplicate records
- Converted date columns to datetime format
- Removed invalid age values
- Validated categorical values
- Saved the cleaned dataset

## Visualizations

- Age Distribution
- Gender Distribution
- Appointment No-show Distribution

## Conclusion

The dataset was successfully cleaned and is now suitable for further analysis and machine learning applications.
