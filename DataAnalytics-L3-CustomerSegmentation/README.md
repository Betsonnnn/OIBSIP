# Customer Segmentation Analysis

## Oasis Infobyte Data Analytics Internship

### Task 3: Customer Segmentation Analysis

---

## 📌 Project Overview

This project was completed as part of the Oasis Infobyte Data Analytics Internship. The objective of this project is to segment customers into different groups based on their annual income and spending score using the K-Means Clustering algorithm. Customer segmentation helps businesses understand customer behavior and improve marketing strategies.

---

## 🎯 Objective

- Analyze customer data.
- Explore customer demographics.
- Perform customer segmentation using K-Means Clustering.
- Identify different customer groups.
- Visualize the customer segments.

---

## 🛠 Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook
- VS Code

---

## 📂 Dataset

The dataset contains **200 customer records** with the following columns:

- CustomerID
- Gender
- Age
- Annual Income (k$)
- Spending Score (1-100)

---

## 📊 Project Workflow

- Import required libraries
- Load the dataset
- Explore the dataset
- Check for missing values
- Perform descriptive statistics
- Select features for clustering
- Apply the Elbow Method
- Train the K-Means model
- Assign cluster labels
- Visualize customer segments
- Interpret the results

---

## 📈 Visualizations

The project includes the following visualizations:

- Annual Income vs Spending Score
- Elbow Method
- Customer Segmentation using K-Means

---

## 🔍 Key Insights

- The dataset contains 200 customers.
- No missing values were found.
- Customers were grouped into five clusters.
- The Elbow Method was used to determine the optimal number of clusters.
- Customers with similar spending behavior and annual income were placed into the same segment.
- Businesses can use these customer groups for personalized marketing and better customer relationship management.

---

## ✅ Conclusion

Customer Segmentation was successfully performed using the K-Means Clustering algorithm. The resulting customer groups provide meaningful insights into customer purchasing behavior and income levels. These insights can help businesses improve marketing campaigns, customer retention, and product recommendations.
